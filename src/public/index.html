Instructions

  1. Add custom API Key and corresponding Cursor Cookie values.
  2. API Key can be added without Cookie, but will have no functionality; you can add Cookie later.
  3. Use custom API Key as the authentication credential for OpenAI API.
  4. The system will automatically poll between multiple Cookies.
    • /v1/models - Model list
    • /v1/chat/completions - Chat completion
